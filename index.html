Description of Group Meeting Activity

The project aims to collect data on renewable energy availability at the GLEAMM facility and develop an API to control workloads on a test cluster's head node.
This involves monitoring available power, planning power usage based on web data, and scheduling jobs on the cluster to match renewable power and backup battery capacity.
The API can also power on/off nodes to minimize energy usage when necessary and extend battery life.
